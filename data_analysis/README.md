Data analysis scripts for the behavioral + eye movement data recorded in the visual search and visibility experiments. <br/>

All .md scripts can be ran on R-Studio to recreate paper figures. <br/>


Aggregated data used in the analysis scripts is located in a single file: `long.csv`, or if you prefer wide format, then `wide.csv`. 

Complete non-aggregated data set can be viewed in  [Dynamic_visual_environments/DFSM/data](https://github.com/herttaleinonen/Dynamic_visual_environments/tree/main/DFSM/data).
